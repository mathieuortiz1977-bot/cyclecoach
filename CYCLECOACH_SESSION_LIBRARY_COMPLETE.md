## Template Coverage by Cycling Goal

### Recreational Cyclist (Long rides, base building)
- Use all BASE templates (17)
- Minimal ANAEROBIC
- Emphasis on aerobic foundation
- **Typical session sequence**: Steady-State → Tempo Pickups → Sweet Spot → Long Ride

### Competitive Racer (Road racing, crits)
- THRESHOLD (14) + ANAEROBIC (8) focus
- Reduced BASE (30% volume)
- Race simulators essential
- **Typical session sequence**: Threshold → VO2 Max → Race Sim → Outdoor Race

### Gran Fondo / Century Rider
- BASE (50% volume) + VO2MAX (20%)
- Minimal ANAEROBIC
- Extended duration emphasis
- **Typical session sequence**: Long Z2 → Sweet Spot → Outdoor 3-4 hr ride

### Mountain Biker (XC)
- VO2MAX + ANAEROBIC + BASE
- Hill repeats critical
- Variable cadence work
- **Typical session sequence**: VO2 Pyramid → Hill Repeats → Outdoor XC
